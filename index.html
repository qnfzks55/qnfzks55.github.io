UI(Front-End) Developer

웹디자인기능생을 하면서 지방기능경기대회 동메달을 입상하여 전국전국기능경기대회에서도 동메달을 입상하고 본격적으로 웹에 재미를 붙여왔습니다.

끝으로, 웹접근성과 UI 를 개발하시는 대한민국 UI개발자들에게 박수를 보냅니다. 감사합니다.
